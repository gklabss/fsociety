# fsociety 🛡️💻
*“Our democracy has been hacked.”*

This is a Python-based cybersecurity toolkit containing various modules for penetration testing and system defense. This project is built for educational purposes and follows the core principles of open-source security tools.

## 📂 Project Structure
- **`attack/`**: Contains scripts for network stress testing (DoS/DDoS).
- **`defense/`**: Tools for system protection, including proxy management and antivirus simulation.
- **`socialengineering/`**: Scripts for testing email-based security awareness.
- **`utilities/`**: Core helper tools like vulnerability scanners and secure password storage.

## 🚀 Getting Started
To use these tools, first clone the repository and install the necessary dependencies.

### Prerequisites
- Python 3.x
- [Git](https://git-scm.com)

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com
   cd fsociety
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
